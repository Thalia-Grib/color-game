# color-game# Color Game 🎨

A fun and interactive Python game built using Tkinter where players must quickly type the **color of the text displayed**, not the actual word itself.

## How It Works
- A word appears on the screen in a random text color.
- Your task is to type the **displayed color** as fast as possible.
- Every correct answer increases your score.
- You have 30 seconds to score as many points as possible.

## Features
- Simple and beginner-friendly GUI
- Real-time score tracking
- Countdown timer
- Randomized colors for each round
- Built with Python and Tkinter

## Technologies Used
- Python
- Tkinter
- Random module

## How to Run
1. Install Python on your computer.
2. Save the script as `colorgame.py`.
